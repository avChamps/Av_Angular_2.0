Screen Diagonal Formula(pythagorean Theorem)

The diagonal of a screen can be calculated using the Pythagorean Theorem: A2 + B2 = C2

Where

  • A is the height of a screen
  • B is the height of a screen
  • C is the diagonal length of a screen
